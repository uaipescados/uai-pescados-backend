# Uai Pescados Backend

Backend inicial em Node.js/Express + PostgreSQL.

## Variáveis de ambiente
- DATABASE_URL
- PORT
- NODE_ENV=production

## Rodar localmente
```bash
npm install
npm start
```

## Endpoints iniciais
- GET /health
- GET /clientes
- POST /clientes
- GET /fornecedores
- POST /fornecedores
- GET /produtos
- POST /produtos
